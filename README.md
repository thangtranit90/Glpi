# GLPI Docker

This is a Docker image for [GLPI](https://glpi-project.org/).

## Requirements

- Docker
- Docker Compose

## Usage

1. Clone this repository
2. Copy `.env.sample` to `.env` and edit the variables
3. Run `bootstrap.sh`
4. Open your browser and go to `http://localhost`
